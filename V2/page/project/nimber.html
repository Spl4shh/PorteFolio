Le jeu du Nimber

Le Projet de langage C consiste à réaliser un jeu de Nim sur un plateau dont on choisi la taille, ce jeu est appelé le Nimber. Le but est de placer le pion dans la case en bas à droite. Cela se complique lorsque les 2 joueurs doivent jouer avec le même pion. Il faudra alors trouver une technique pour gagner, créer une IA pour jouer contre le joueur.
